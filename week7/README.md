# Лабораторна робота: Lag Compensation  Lag Rewind

 
---

## 1. Що було зроблено 

* **Буфер історії хитбоксів:** Сервер кожні 0.05 сек зберігає знімки (snapshots) позицій голови та торса кожного персонажа за останню 1 секунду.
* **Серверний RPC з валідацією:** Створено функцію `ServerFireWeapon`, яка приймає час пострілу від клієнта (`ClientTimeStamp`). Реалізовано захист `_Validate`: якщо затримка клієнта перевищує **250 мс**, сервер розриває з'єднання для захисту від чітів.
* **Перемотка та інтерполяція часу:** При пострілі сервер знаходить два найближчих знімки в історії, інтерполює точні координати хитбоксів на момент пострілу і проводить перевірку променем (`LineTrace`).
* **Дебаг-візуалізація:** Додано консольну команду `vfx.DrawRewindDebug 1`. Вона малює сфери хитбоксів у минулому часі: **зелені** - при влучанні, **червоні** - при промаху.
* **Авто-патруль цілі:** Створено  рух мішені по колу для точності тестів.
